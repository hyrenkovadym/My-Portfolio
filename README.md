# My Portfolio Platform

Full-stack portfolio platform demonstrating backend and frontend architecture.

## Stack

Backend
- NestJS
- Prisma
- PostgreSQL
- JWT Auth

Frontend
- Next.js
- React
- TypeScript

## Features

- Authentication (JWT)
- Product API
- Category API
- Admin product management
- Docker infrastructure

## Run project

docker compose up -d

pnpm install
pnpm dev
